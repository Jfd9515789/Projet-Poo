Pour la base de données, il faut soit charger les fixtures, mais il faudra rajouter des commentaires manuellement dans la
base de données. Ou alors importer la base de données (dans le dossier) directement dans laquelle j'ai déjà associé manuellement 5 commentaires aux peintures.

Pour afficher une peinture en détail elle doit avoir un commentaire.

-On ne peut pas non plus supprimer une peinture ayant un commentaire associé à elle (il faut d'abord supprimer le commentaire en base de données), ensuite supprimer la peinture en question.

Sinon l'ajout et la modification sont fonctionnels sans contraintes.
